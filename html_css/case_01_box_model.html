Box Model

My box

CSS styles:

  #my-box {
    background-color: #ccc;
    width: 100px;
    height: 100px;
    padding: 10px;
    border: 5px solid blue;
    margin: 10px;
  }

W3 standard:

  box width = width + 2 * (padding + border)
            = 100 + 2 * (10 + 5)
            = 130

IE:

  box width = width
            = 100
